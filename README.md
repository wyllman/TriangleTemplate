TriangleTemplate  
================  

Template web html5, css3 y javascript.  
Basado en ["Triangle Free CSS Template"](http://preview.astemplates.com/itempreview/538), autor: [themeum](https://www.themeum.com/)  

Recursos  
--------  
  
BootStrap 4.2.1  
Jquery 3.3.1  
Font-Awesome 5.6.3  
Animate.css 3.7.0  
WOW 1.3.0
